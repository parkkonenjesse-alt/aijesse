# Esimerkit

Mitä asiakas voi rakentaa koulutuksen aikana. Konkreettisia, arjessa hyödyllisiä ratkaisuja.

- **Tarjouslaskuri**, joka tekee luonnoksen minuuteissa.
- **Asiakaspalvelun avustaja** yleisimpiin kysymyksiin.
- **Raporttien ja palautteiden** automaattinen kooste.
- **Sisällöntuotanto** somen ja uutiskirjeen pohjiksi.
- **Datan siirto ja siivous** järjestelmien välillä.
- **Sisäinen hakutyökalu** omiin dokumentteihin.

Nämä eivät ole kiinteä lista vaan esimerkkejä. Varsinainen rakennettava valitaan asiakkaan arjen mukaan kartoituksessa ([prosessi.md](prosessi.md)).
