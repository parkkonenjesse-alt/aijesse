# Hinnasto

Hinnat alv 0 % (+ alv päälle). Voimassa toistaiseksi. Tämä tiedosto on hintojen totuuden lähde: päivitä tänne ensin, sitten nettisivulle.

## Startti

- **1 900 € + alv**
- Yhden päivän työpaja
- Koko tiimi pääsee alkuun ja vie ensimmäisen automaation käyttöön saman päivän aikana.

## Rakennusjakso (suosituin)

- **8 900 € + alv**
- Neljän viikon ohjelma
- Valitaan yksi oikea ongelma ja rakennetaan ratkaisu tuotantoon asti.

## Kumppani

- **alk. 1 500 € / kk + alv**
- Jatkuva kuukausimalli
- Pidän tiimin kehityksessä mukana. Uudet ratkaisut ja tuki silloin kun tarvitsette.
