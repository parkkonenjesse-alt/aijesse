# Kohderyhmä

## Kenelle

Pk-yrityksille, jotka haluavat ottaa tekoälyn oikeasti käyttöön, eivät vain puhua siitä. Ei vaadi teknistä taustaa, sopii jokaiselle oman työnsä kautta.

## Roolit / osastot

- **Myynti** — tarjoukset, liidien käsittely, asiakasviestintä
- **Toimisto** — raportit, yhteenvedot, dokumenttien käsittely
- **Tuotanto** — datan siirto ja siivous, sisäiset työkalut
- **Johto** — koosteet, päätöksenteon tuki

## Hyvä asiakas

- Haluaa kyvyn jäävän taloon, ei vain valmista ratkaisua.
- Valmis varaamaan tiiminsä aikaa hands-on-rakentamiseen.
- Tunnistaa edes yhden toistuvan, työlään prosessin jonka haluaa kuntoon.
