# Myyntimateriaalit

Käytännön myyntikitti yhden hengen yritykselle: viestit, keskustelurunko, vastaväitteet, tarjous ja LinkedIn. Kaikki johdettu tietopankin totuuden lähteestä (paketit [palvelut.md](../palvelut.md), hinnat [hinnasto.md](../hinnasto.md), äänensävy [brandi-ja-viestinta.md](../brandi-ja-viestinta.md)). Jos paketti tai hinta muuttuu, päivitä ensin tietopankki, sitten nämä.

## Äänensävy myynnissä (tärkeä erottelu)

- **Nettisivu = brändiproosa, passiivi** ("rakennetaan", "käydään läpi"). Ei "minä", ei teennäistä "me". Ks. PÄÄTÖS 26.6. [brandi-ja-viestinta.md](../brandi-ja-viestinta.md).
- **Myyntiviestit ja LinkedIn = Jesse henkilönä, minä-muoto.** Sähköposti ja viesti on aitoa ihmisten välistä yhteydenottoa, joten "minä" on luontevaa ja uskottavaa. Yhden hengen yrityksessä "me" kuulostaisi ontolta. Tämä on tarkoituksellinen ero, ei rikkomus.

Yhteinen molemmille: rauhallinen, konkreettinen, ei hypeä, ei huutomerkkejä, ei em-dasheja, ei koristenumeroita, sentence case.

## Yksi tavoite kaikessa

Jokainen materiaali ohjaa samaan ensiaskeleeseen: **30 minuutin maksuton keskustelu**, jossa katsotaan mistä kannattaa aloittaa. Ei sitoumuksia. Sama sana läpi: "Varaa keskustelu" → "Varattu".
